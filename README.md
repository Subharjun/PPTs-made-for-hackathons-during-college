## 💻 How to View & Download

Because pitch decks can be large and binary files are hard to preview on GitHub, I built a custom **hacker-style terminal downloader** using Bash.

### Using the Interactive Downloader (Linux / macOS / WSL)

You can run the interactive terminal script to browse and download the presentations safely to your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Subharjun/PPTs-made-for-hackathons-during-college.git
   cd PPTs-made-for-hackathons-during-college
   ```

2. **Make the script executable:**
   ```bash
   chmod +x fetch-pitches.sh
   ```

3. **Run the interactive downloader:**
   ```bash
   ./fetch-pitches.sh
   ```

You will see a colorful ASCII menu where you can press a number to download a specific pitch deck or type `A` to download them all!

*(Requires `curl` or `wget` to be installed on your system).*
